# 🛒 E-Commerce Sales Analysis

A Python-based data analysis project for exploring, processing, and visualizing e-commerce sales data. The project includes data preprocessing scripts, analytical reports, and a Power BI dashboard for business insights.

---

## 📁 Project Structure

```
E-commerce-sales/
├── data/               # Raw and processed datasets
├── src/                # Python scripts for data processing and analysis
├── powerbi/            # Power BI dashboard files (.pbix)
├── reports/            # Generated reports and output files
└── requirements.txt    # Python dependencies
```

---

## 🔍 Overview

This project analyzes e-commerce sales data to uncover trends, measure performance, and support data-driven decision-making. It combines Python for data wrangling and analysis with Power BI for interactive visual reporting.

Key areas of analysis include:

- Sales trends over time
- Product and category performance
- Customer behavior insights
- Revenue and profitability metrics
- Regional or segment-level breakdowns

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Data processing & analysis |
| Pandas | Data manipulation |
| Power BI | Interactive dashboard & visualization |

## ⚙️ Getting Started

### Prerequisites

- Python 3.8+
- Power BI Desktop (for viewing the `.pbix` dashboard)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/sanaa-zeroual/E-commerce-sales.git
cd E-commerce-sales
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the analysis scripts:

```bash
python src/<script_name>.py
```

---

## 📊 Power BI Dashboard

The `powerbi/` folder contains the Power BI dashboard file. To view it:

1. Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
2. Open the `.pbix` file from the `powerbi/` folder
3. Explore the interactive visuals and filters
